# Synaphack (Frontend)

This repository contains the frontend for the Synaphack hackathon platform built with Next.js and TypeScript.

Repository (new): https://github.com/gaurangkgd/TechSprint.git

Overview
- Framework: Next.js 14
- Language: TypeScript
- Styling: Tailwind CSS
- Firebase: Auth, Firestore, Storage
- Integrations: Cloudinary, Email service, Git MCP (AI code analysis)

Quick start

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```
